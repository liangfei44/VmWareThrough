Name

VixDiskLib_Exit

Description

void
VixDiskLib_Exit(void);

This function releases all resources held by VixDiskLib.

Return Value

None.

Remarks

  • Every VixDiskLib_Init() must have a matching call to VixDiskLib_Exit().

Example

   VixDiskLib_Exit();
